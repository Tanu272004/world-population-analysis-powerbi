# 🌍 World Population Trends & Insights (1960–2024)

## 📌 Project Overview
This project is developed as part of my **Data Science Internship (Task 01) at Prodigy InfoTech**.  
The goal is to analyze **World Bank Population Data (1960–2024)** and create an **interactive Power BI dashboard** to extract insights on global population growth.

---

## 📊 Dashboard Features
- **Global Population Growth (1960–2024)** – Line chart
- **Top 10 Most Populated Countries** – Horizontal bar chart
- **Region-Wise Population Analysis** – Donut chart
- **Population Growth % (1960–2024)** – Line chart
- **Key KPIs** – Total Population, Growth %, Top 3 Regions

---

## 📂 Files in Repository
- `World_Population_Analysis.pbix` → Power BI dashboard file
- `world_population_data.csv` → Dataset used
- `README.md` → Project documentation
- `Task01_Report.pdf` → Internship task report

---

## 🔑 Key Insights
- Population has grown consistently since 1960.
- East Asia & Pacific region dominates in share.
- IDA & IBRD categories and Low & Middle Income groups are top contributors.
- Growth % trend varies regionally, showing demographic shifts.

---

## 🚀 Tools Used
- **Power BI** – Dashboard & Data Visualization
- **World Bank Data** – Dataset Source

## 🏁 Conclusion
This dashboard provides a comprehensive view of population growth trends and can be leveraged for **policy-making, research, and academic studies**.
